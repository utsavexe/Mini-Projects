# 🚀 Mini Projects Collection

This repository contains a collection of beginner-friendly **web development mini projects** built using HTML, CSS, JavaScript, and TypeScript. These projects are perfect for learning, practice, and showcasing fundamental frontend skills.

---

## 📂 Projects Included

### 📝 Note App
- Create, edit, and delete notes
- Built with TypeScript + Vite
- Local storage support

### 📷 QR Code Reader
- Scan QR codes using camera
- Progressive Web App (PWA) support

### ✊ Rock Paper Scissors Game
- Interactive UI with sound effects
- Play against computer

### ⏱️ Stopwatch App
- Start, stop, reset functionality
- Simple and responsive design

### 🌦️ Weather App
- Fetches real-time weather data
- Clean UI with API integration

### 🧮 Calculator Web App
- Basic arithmetic operations
- Minimal and user-friendly interface

### ❓ Quiz App
- Multiple choice questions
- Score tracking system

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- TypeScript
- Vite

## ⚙️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Mini-Projects.git
   ```

2. Navigate to project folder:
   ```bash
   cd Mini-Projects
   ```

3. Open any project folder and run:
   ```bash
   npm install
   npm run dev
   ```
   *(Only required for TypeScript/Vite projects like Note App)*

4. For simple projects:
   - Open `index.html` directly in browser
